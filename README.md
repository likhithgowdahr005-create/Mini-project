## Description
This project demonstrates a real-time hologram visualization using a device camera.
It mirrors and tiles the video feed to simulate a holographic display effect.

## Live Demo
https://hologramview.netlify.app/

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Web Camera API
- MediaPipe (via CDN)
- Netlify (Deployment)

## How to Run
1. Open index.html in a browser
2. Allow camera access
3. Use on-screen controls to calibrate hologram view
